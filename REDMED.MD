# 项目背景

项目很简单，全部内容都融合在index.html中，是一个自定义的名叫SBTI测试。

# 需求背景

现有的SBTI测试，一共27种结果，都在TYPE_LIBRARY里。

我想再拓展一下，结合一下星座。

# 实现步骤

1、先了解index.html的结构和逻辑

2、检索一下各个星座的特征，要求至少200字。

3、27种结果，融合星座的特征，按照原本结果的语言风格，生成12*27种融合分析。这里可以多agent协同

4、目前我们除了之前的结果集合TYPE_LIBRARY之外，应该还有星座特征集合、星座+SBTI结果集合。

5、修改源码，在测试开始前，选择自己的星座，也可以跳过星座，直接测试。

6、修改源码，测试结束后，如果选择了星座，输出：原本结果 + 星座特征 + 融合分析。如果没选择星座，则输出结果保持和初版一样。

# 注意

1、不要修改项目的整体，不要修改测试题

2、仅仅从结果上，改变项目
